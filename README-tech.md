# Develop


## Back-End Lint

```bash
cd /[your-path]/anchor-nft-ticketing
```

Format the code:

```bash
cargo fmt
```

Lint :

```bash
cargo clippy
```


## Front-End Lint

```bash
cd /[your-path]/anchor-nft-ticketing
```

Lint and format the code :

```bash
npm run lint:fix
```


## Run Localnet Validator

```bash
cd /[your-path]/anchor-nft-ticketing
```

```bash
npm run ledger
```


## Run test with Localnet

```bash
cd /[your-path]/anchor-nft-ticketing
```

```bash
npm run test
```
